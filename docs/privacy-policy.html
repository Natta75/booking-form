Политика конфиденциальности

Дата вступления в силу: 09 декабря 2025 г.

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных, предоставляемых пользователями при использовании формы бронирования на нашем веб-сайте.

Обработка персональных данных осуществляется в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных».

3. Цель обработки персональных данных

Персональные данные обрабатываются в следующих целях:

  • Обработка заявок на бронирование услуг
  • Связь с пользователями для подтверждения бронирования
  • Предоставление информации о статусе заявки
  • Улучшение качества обслуживания

4. Категории обрабатываемых персональных данных

Мы собираем и обрабатываем следующие категории персональных данных:

  • Имя — для персонализированного обращения
  • Номер телефона — для связи с пользователем
  • Адрес электронной почты — для отправки подтверждений и уведомлений
  • Дата бронирования — для обработки заявки
  • IP-адрес — для обеспечения безопасности и предотвращения мошенничества

5. Правовое основание обработки

Правовым основанием обработки персональных данных является согласие пользователя, предоставленное путем заполнения формы и установки соответствующего чекбокса.

Важно: Предоставление персональных данных является добровольным. Однако отказ от предоставления данных сделает невозможным обработку вашей заявки на бронирование.

6. Способы и сроки обработки персональных данных

6.1. Способы обработки

Персональные данные обрабатываются следующими способами:

  • Сбор, запись, систематизация, накопление
  • Хранение, уточнение (обновление, изменение)
  • Извлечение, использование
  • Передача (распространение, предоставление, доступ)
  • Блокирование, удаление, уничтожение

6.2. Срок хранения

Персональные данные хранятся в течение 3 (трех) лет с момента последнего взаимодействия с пользователем, после чего подлежат удалению или обезличиванию.

7. Передача персональных данных третьим лицам

Для обеспечения функционирования системы бронирования мы используем следующие сервисы:

  • Google Sheets (Google LLC) — для хранения данных заявок. Политика конфиденциальности: https://policies.google.com/privacy
  • Telegram (Telegram Messenger Inc.) — для отправки уведомлений оператору. Политика конфиденциальности: https://telegram.org/privacy
  • Google reCAPTCHA (Google LLC) — для защиты от спама и ботов. Политика конфиденциальности: https://policies.google.com/privacy

Передача данных осуществляется только в объеме, необходимом для выполнения указанных целей. Все третьи лица обязаны обеспечивать конфиденциальность и безопасность персональных данных.

8. Меры по защите персональных данных

Мы применяем следующие технические и организационные меры защиты:

  • SSL/TLS шифрование (HTTPS) — для защиты данных при передаче
  • Серверная валидация — для предотвращения некорректных данных
  • Защита от DDoS-атак — rate limiting и другие механизмы
  • Ограничение доступа — доступ к данным имеют только уполномоченные лица
  • Логирование действий — для аудита и расследования инцидентов
  • Регулярное обновление ПО — для устранения уязвимостей

9. Права субъектов персональных данных

В соответствии с 152-ФЗ вы имеете следующие права:

  • Право на информацию — получение сведений об обработке ваших данных
  • Право на доступ — получение копии ваших персональных данных
  • Право на исправление — уточнение неверных или неполных данных
  • Право на удаление — требование удалить ваши данные (право на забвение)
  • Право на отзыв согласия — отозвать согласие на обработку данных в любое время
  • Право на обжалование — обжалование действий оператора в Роскомнадзоре
